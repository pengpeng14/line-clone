# Line clone Web application

> A React project

## Technology
  - React
  - Material UI
  - Amplify library (Cognito, S3, AppSync, DynamoDB, GraphQL)
  - Firestore
  - WebRTC

## Build Setup
```bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm start
```
